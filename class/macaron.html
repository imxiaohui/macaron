The Macaron related classes¶

Warning

Generally, these classes should not be used by user directly.

class macaron.Macaron¶

Macaron controller class. Do not instance this class by user.

connection¶

dict object holds sqlite3.Connection

get_connection(meta_obj)¶

Returns Connection and adds reference to the object which uses it.

class macaron.TableMetaInfo(conn, table_name, cls)¶

Table information class. This object has table information, which is set to ModelClass._meta by ModelMeta. If you use Bookmark class, you can access the table information with Bookmark._meta.
